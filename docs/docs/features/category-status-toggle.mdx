# Category Status Toggle Feature

## Overview
Added a status badge to the category list to visually display the active/inactive status of categories.

## Implementation Details
- Updated the category list template to show status badges
- Active categories display a green "Active" badge
- Inactive categories display a grey "Inactive" badge
- No changes to underlying data model or business logic

## Usage
The status badge is automatically displayed in the category list table for each category.

## Requirements
- Requires Django template rendering
- No additional dependencies
