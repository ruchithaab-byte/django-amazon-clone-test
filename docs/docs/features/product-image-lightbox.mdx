# Product Image Lightbox Functionality

## Overview
The product list view now supports a click-to-enlarge image lightbox feature for product images.

## Features
- Clickable product images with hover effect
- Full-screen modal for image viewing
- Responsive and centered image display
- Modal can be closed via X button or clicking outside

## Usage
When viewing the product list, click on any product image to open a full-screen modal displaying the image with its product name.

## Technical Implementation
- Uses Bootstrap modal for lightbox functionality
- Implements event delegation for dynamic image handling
- Provides smooth user experience with hover and transition effects
