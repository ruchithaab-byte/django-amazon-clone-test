## Exporting Product List

### Export to CSV

The product list page now supports exporting the entire product list to a CSV file. This feature allows easy data extraction for reporting and analysis.

#### How to Export

1. Navigate to the Product List page
2. Click the "Export to CSV" button located next to the search bar
3. The CSV file will download automatically

#### Exported Data
The CSV file includes the following product details:
- Product ID
- Name
- Brand
- Category
- Subcategory
- Maximum Price
- Discount Price
- Stock Quantity
- Creation Date

#### Filtering and Ordering
- Any active search filters or ordering will be preserved in the exported CSV
- Allows for targeted exports based on current view

### Benefits
- Easy data extraction
- Compatible with spreadsheet software
- Supports reporting and inventory management
