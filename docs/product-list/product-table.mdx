# Product List Table

## Overview
The Product List table provides a comprehensive view of all products in the system. 

## Features
- Displays product details including ID, Name, Subcategory, URL Slug, Status, and Creation Date
- "Actions" column with quick access buttons
  - "View" button to edit product details
  - "Add Media" button to manage product images and media

## Table Columns
| Column | Description |
|--------|-------------|
| ID | Unique identifier for the product |
| Name | Product name |
| Subcategory | Category the product belongs to |
| URL Slug | SEO-friendly URL identifier |
| Status | Product active/inactive state |
| Created At | Timestamp of product creation |
| Actions | Quick action buttons |

## Action Buttons
- **View**: Navigate directly to product edit page
- **Add Media**: Upload and manage product images

## Notes
- Table supports pagination for large product lists
- Search functionality available to filter products
