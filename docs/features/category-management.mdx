## Category and Subcategory List Filter Functionality

### Clear Filter Button

In both the Category and Subcategory list pages, a "Clear" button has been implemented next to the search input field. This button provides a quick way to reset the filter and show all items.

#### Features
- Appears in both Category and Subcategory list pages
- Located next to the search input field
- Instantly clears any applied filters
- Redirects to the list page without any filter parameters

#### Usage
1. Apply a filter by entering text in the search input
2. Click the "Clear" button to reset the filter
3. The page will refresh, showing all items

### Implementation Details
- Button is styled with Bootstrap's `btn-secondary` class
- Uses Django's URL routing to reset the view
- No additional backend logic required
