## Bulk Product Selection

The product list page now supports bulk selection of products:

### Features
- A "Select All" checkbox in the table header
- Individual checkboxes for each product row
- Easy toggle to check/uncheck all products

### Usage
1. Click the checkbox in the table header to select/deselect all products
2. Individual product checkboxes can also be selected manually
3. Useful for performing bulk actions on multiple products

### Technical Implementation
- JavaScript function `toggleAllProducts()` handles the select all functionality
- Implemented in the product list template with minimal overhead
