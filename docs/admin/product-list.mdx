# Product List Admin Page

The Product List page in the admin panel provides a comprehensive view of all products in the system.

## Features

### Product Display
Each product card displays:
- **Product Name**: Shown in a badge at the top
- **Subcategory**: The category the product belongs to
- **URL Slug**: The unique URL identifier for the product
- **Created At**: The date and time when the product was created (formatted as YYYY-MM-DD HH:MM)
- **Active Status**: Toggle showing whether the product is active

### Sorting Options
Products can be sorted by:
- **ID**: Sort by product ID
- **Title**: Sort alphabetically by product name
- **Description**: Sort by product description
- **Created At**: Sort by creation date (newest or oldest)

### Search
Use the search bar to filter products by name or other attributes.

### Actions
Each product card provides quick actions:
- **Edit**: Modify product details
- **Add Media**: Upload new media for the product
- **Edit Media**: Modify existing product media
- **Add Stocks**: Update product inventory

## Usage Example

To access the product list:
1. Navigate to the admin panel
2. Click on "Product List" in the navigation menu
3. Use the "Sort By" links to organize products
4. Use the search bar to find specific products
