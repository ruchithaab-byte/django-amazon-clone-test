## Product List Functionality

### Bulk Product Selection

The product list page now includes a "Select All" checkbox functionality to enable bulk selection of products.

#### Features
- A checkbox in the table header allows selecting/deselecting all products
- Individual checkboxes for each product row
- JavaScript-powered toggle mechanism

#### Usage
1. Click the checkbox in the table header to select/deselect all products
2. Individual product checkboxes can be selected independently
3. The "Select All" checkbox state reflects the current selection of individual checkboxes

**Note:** This feature facilitates easier management of multiple products in the admin interface.
