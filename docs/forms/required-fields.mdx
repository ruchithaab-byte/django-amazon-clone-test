
# Required Field Indicators

## Overview
All create and edit forms in the admin templates now include visual indicators for required fields.

## Implementation Details
- Red asterisk (*) added to label for required fields
- Implemented using Django template conditionals
- Styling applied with inline CSS for red color

### Example Template Logic
```html
<label>
  {{ field.label }}
  {% if field.field.required %} 
    <span style="color: red;">*</span>
  {% endif %}
</label>
```

## Affected Templates
- Category Create/Update
- Sub-Category Create/Update
- Product Create/Edit
